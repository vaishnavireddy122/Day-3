# Day-3

<h3>JS OBJECTS :</h3>
<p>
A JavaScript object is a collection of key-value pairs. Each key (also called a property name) is a string, and its value can be any data type.
</p>
<pre>
  const person = {
  name: "Alice",
  age: 30,
  isStudent: false,
  hobbies: ["reading", "cycling"],
  greet: function () {
    console.log("Hello!");
  }
};

</pre>
